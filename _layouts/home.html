Work and Projects

OpenDNS

{% image opendns.png %}

OpenDNS provides cloud-delivered network security for any device, anywhere. On the Research Systems team, I am currently building out a graph database to record, analyze, and enforce all security-related events. This system will also be used to improve the efficacy and velocity of our researchers. Common tools include Python, Java, HBase, and Kafka.

OpenLate

{% image openlate.jpg %}

OpenLate is an engineering meetup group that I run in partnership with OpenDNS. Events are held biweekly and feature a talk held by a technical thought leader. In its first year, I helped grow OpenLate to over 1,000 members, which also resulted in 3 technical hires and over $100K worth of deals.

StaffJoy

{% image staffjoy.png %}

StaffJoy is an application I am building that aims to makes teams more efficient by automating workforce scheduling. At the core of this project is a solution for the nurse scheduling problem written in Julia, which can schedule a workforce in under 1 minute of computation time. This tool is still in the early stages of development, please stay tuned for more details.
